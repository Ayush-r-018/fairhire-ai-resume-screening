# FairHire - AI-Based Anti-Bias Resume Screening System

## Project Overview

FairHire is a Minor Project that aims to reduce bias in the resume screening process using Artificial Intelligence and Natural Language Processing (NLP).

The system compares job descriptions and candidate resumes using Sentence-BERT (SBERT) embeddings and cosine similarity to identify the most relevant candidates based on skills, experience, and semantic meaning rather than simple keyword matching.

## Features

- Resume Matching using SBERT
- Semantic Similarity Scoring
- Resume Ranking
- Bias Reduction
